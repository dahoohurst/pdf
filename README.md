How to Install
--------------------------------------------------------------------------------
Installation via composer is easy:

  "repositories": [
      {
          "type": "vcs",
          "url": "https://github.com/daniel-hocking/pdf"
      }
  ],
  "require": {
  ...
    "gears-update/pdf": "dev-master"
  }
