How to Install
--------------------------------------------------------------------------------
Installation via composer is easy:

  "repositories": [
      {
          "type": "vcs",
          "url": "https://github.com/hejingyuan199/pdf"
      }
  ],
  "require": {
  ...
    "gears/pdf": "dev-master"
  }
